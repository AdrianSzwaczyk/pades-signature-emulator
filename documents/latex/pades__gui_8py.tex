\doxysubsection{Detailed Description}
GUI for signing and verifying PDF files using RSA with PADES standard. 

This GUI application allows users to\+:
\begin{DoxyItemize}
\item Sign PDF documents using a private key stored on a USB drive.
\item Verify signed PDFs using a public RSA key. There are two modes\+: signing and verifying. The signing mode requires a PIN for the private key, while the verifying mode requires a public key file. It uses AES + PBKDF2 for private key decryption and SHA256 + RSA for digital signing. ~\newline
 Used libraries\+:
\item \textquotesingle{}tkinter\textquotesingle{} for GUI components
\item \textquotesingle{}pypdf\textquotesingle{} for PDF manipulation
\item \textquotesingle{}pycryptodome\textquotesingle{} for cryptographic operations
\item \textquotesingle{}os\textquotesingle{} for file handling
\item \textquotesingle{}threading\textquotesingle{} for threads working in the background Functions from other modules like sign\+\_\+pdf(), decrypt\+\_\+private\+\_\+key(), and verify\+\_\+pdf\+\_\+signature() are used for signing, decrypting, and verifying operations. 
\end{DoxyItemize}
