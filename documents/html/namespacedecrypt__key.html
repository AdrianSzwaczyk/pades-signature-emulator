Decrypts an RSA private key stored on a USB device using AES and a PIN-based key derivation.

This function looks for an encrypted private key file inside the "keys" folder on the specified in params USB path.
It reads the encrypted data and derives the AES key from the user-provided PIN using PBKDF2 with a 128-bit salt and 600,000 iterations.
Decryption is done using AES in EAX mode. Progress updates are reported with the callback function.

The function uses the following libraries:

  • 'pycryptodome' – Crypto.Protocol.KDF.PBKDF2, Crypto.Cipher.AES
  • 'os' – to search for the encrypted file on the USB drive
Parameters
usb_path(str): The path to the USB device where the encrypted private key is stored.
pin(str): The PIN used to derive the decryption key.
progress_callbackCallable[[str], None]: A callback function to show decryption progress.
Returns
bytes | None: The decrypted private key (in bytes) if successful, or None if an error occurs or the key file is missing.
