{\footnotesize\ttfamily pades\+\_\+utils.\+update\+\_\+usb\+\_\+devices (\begin{DoxyParamCaption}\item[{}]{usb\+\_\+path\+\_\+entry }\end{DoxyParamCaption})}



Allows to update the list of USB devices currently connected to the computer. 

It auto-\/selects one containing a private key.

It makes sure that if there is a USB device connected with encrypted private key, it will be selected by default in the GUI and if there are no USB devices connected, the box will be empty. It uses the \doxylink{namespacepades__utils_a812f59718745ff848df65ae4d9ab167c}{get\+\_\+usb\+\_\+devices()} function to retrieve the list of USB devices and the \doxylink{namespacepades__utils_a4a1d91e885b30d89eab305fb65993991}{find\+\_\+usb\+\_\+with\+\_\+key()} function to filter them and to get path of a device with a private key.
