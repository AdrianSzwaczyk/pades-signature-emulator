Encrypts a private key using AES symetric encryption, where the encryption key is the user-provided PIN using PBKDF2 (Password-Based Key Derivation Function 2).

The resulting encrypted output is composed of: salt (16 bytes) + nonce (16 bytes) + authentication tag (16 bytes) + ciphertext (N bytes).

Parameters
private_keyThe private key to encrypt (bytes).
pinThe password used to derive the encryption key.
progress_callbackA callback function to update the GUI progress bar.
Returns
The encrypted private key as bytes, or None if an error occurs.
