key_generation.generate_rsa_keys ( )

Generating two RSA keys: a private key and a public key (4096 bits).

Both keys are stored in PEM format in bytes format.

Returns
A tuple containing the private key and public key in bytes format. If an error occurs, returned tuple is (None, None).
