\doxysubsection{Detailed Description}
This file contains the GUI for the RSA key generation application. 

This application contains a graphical user interface (GUI) that allows users to generate RSA keys, encrypt the private key with a PIN, and save the keys to a USB device and a specified public key path. The GUI is built using Tkinter.
